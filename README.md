# 停车场管理系统 java727

## 项目概述

停车场管理系统是一款基于Spring Boot框架和Spring MVC架构的软件，前端采用VUE技术。该系统主要支持用户和管理员两种角色，能够高效地处理停车场相关的各项业务。

## 技术栈

- 后端：Java、Spring Boot、Spring MVC
- 前端：VUE
- 数据库：MySQL

## 功能模块

系统主要分为以下几个功能模块：

- 用户模块
- 管理员模块
- 停车场管理
- 账单管理
- 车位预订

## 系统角色

- 用户：注册、登录、预订车位、查询账单等
- 管理员：管理用户信息、车位信息、账单信息等

## 运行环境

- Java环境：JDK 1.8及以上
- 数据库：MySQL 5.7及以上
- IDE：IntelliJ IDEA

## 部署步骤

1. 下载代码到本地
2. 导入数据库SQL文件到MySQL数据库
3. 使用IntelliJ IDEA打开项目
4. 运行启动程序
